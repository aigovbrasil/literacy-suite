# Praxis OS · Professional Services Operating System

**Category:** Major OS · **Version:** current · **Language:** EN / PT-BR
**Owner:** Leonardo Batista

## What it does
A complete operating system for consultants and knowledge-worker operators. Routes tasks across 23 specialist agents in 5 phases: SIGNAL → INTELLIGENCE → ARTIFACT → DELIVERY → FEEDBACK. Manages the full consulting lifecycle from signal detection to client delivery.

## Activate when
```
"/praxis"              any "/praxis-*" command
"new client just signed"        "where am I in the case"
"what's the next step"          project already has praxis-os state files
(consultant_config.yaml · session_state.yaml · manifest.yaml)
```

## 5 phases · 23 agents
```
SIGNAL       → detect and capture business signals
INTELLIGENCE → research, analyze, score
ARTIFACT     → generate deliverables
DELIVERY     → client-facing output
FEEDBACK     → measure, learn, improve
```

## Rules
- Never invents commands
- Never promotes hypotheses to facts
- Always confirms before destructive action
- Routes to specialists — does not wrap them unnecessarily
