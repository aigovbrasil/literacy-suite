# Empower V4 · AI Usage Evaluator

**Category:** Major OS · **Version:** 4.0 · **Language:** EN
**Owner:** Leonardo Batista

## What it does
Evaluates, normalizes, compares, and improves AI/Claude usage patterns. Converts conversation exports into normalized datasets, computes structural maturity scores, detects weak prompt patterns, compares accounts, and produces publication-ready reports.

Designed for **system-level evaluation** — not isolated prompt editing.

## Activate when
```
"Run Empower"                    "Empower V4"
"analyze Claude conversations"   "compare accounts"
"calculate maturity score"       "detect prompt risk patterns"
"normalize interaction data"     "generate usage quality report"
"build quality-label queue"      "publishable AI usage analysis"
```

## What it produces
- Normalized interaction dataset (canonical schema)
- Maturity score per account/user
- Prompt risk cluster map
- Quality lift analysis
- GitHub / Medium / LinkedIn report-ready output

## Input formats
XLSX · CSV · JSON · JSONL · ZIP conversation exports

## Do NOT activate for
One-off prompt rewriting · generic Claude questions · advice without historical data
