# FORGE Visual Canvas · Premium Visual Artifacts

**Category:** Major OS · **Version:** 2.0.0 · **Language:** PT-BR / EN
**Owner:** Leonardo Batista + FORGE

## What it does
Generates premium visual artifacts (HTML, React/JSX, SVG, PDF A4, PPTX) with brand consistency, dark/light mode, and runtime visual language switching. Anti-"AI slop" by design — no purple gradients, no generic Inter font.

**3-layer pipeline:** Structure (format) → Tokens (brand) → Acabamento (mood/visual language)

## Activate when
```
"criar artifact"          "gerar showroom"         "FORGE"
"visual canvas studio"    "página de venda"         "apresentação executiva"
"ebook interativo"        "diagrama profissional"   "PDF estratégico A4"
"modo dark e light"       pixel-perfect UI clone    "estilo McKinsey/Stripe/Linear"
```

Also activates for any of the 10 catalogued visual languages.

## 10 visual languages
Executive · SaaS · Editorial · Dashboard · Terminal · Government · Consulting · Startup · Data · Minimal

## Do NOT activate for
Pure text tasks · backend code · conceptual questions without visual deliverable
