# x-ray-skill-packager · Suite Distribution Package

**Category:** X-Ray Suite · **Version:** 1.0.0 · **Language:** PT-BR
**Owner:** Leonardo Batista · **License:** X-Ray Commercial Suite

## What it does
Packages the complete X-Ray suite (all 8+ skills) into a single distributable ZIP ready for delivery to a new consultant buyer. Includes all SKILL.md files, templates, README, and setup instructions.

## Activate when
```
"empacotar suite X-Ray"      "gerar ZIP da suite"
"package X-Ray"              "preparar entrega para consultor"
"/x-ray-package"             "build da suite"
"criar arquivo de distribuição" "pacote para venda"
"ZIP para upload no Claude.ai"
```
Also activates when delivering the suite to a new consultant or redistributing an updated version.

## Do NOT activate for
Packaging a single skill (use `skill-creator` + `cmd-02-mirp`)

## Output
Single ZIP: all skills + README + setup instructions + brand config template
