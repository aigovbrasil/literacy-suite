# x-ray-abs · Strategic Business Diagnosis

**Category:** X-Ray Suite · **Version:** 3.0.0 · **Language:** PT-BR / EN  
**Owner:** Leonardo Batista · **License:** X-Ray Commercial Suite

---

## What it does

Runs a structured strategic diagnosis on any business, product, or idea. Produces a scored gap map, a Mermaid system diagram, a Decision Question, and an executive synthesis — all with epistemic labels (fact vs hypothesis vs inference).

---

## Activate when

```
"analise esse negócio"    "diagnóstico"        "raio-x"
"x-ray"                   "o que está faltando" "gaps"
"avalie esse modelo"      "roadmap"             "SWOT"
"proposta comercial"      "priorize com score"  "audite essa conversa"
```

Also activates on document upload when strategic analysis is implied.

**v3 extras:** `"qual fase estou"` · `"qual gate"` · `"deliverable Sxx"` → Canonical Engine (19 phases × 13 gates). `"certeza calibrada"` · `"#arede"` → Uncertainty Mode (AREDE).

---

## 3-step quickstart

```
1. Paste business context, briefing, or research notes
2. Say: "x-ray" or "analise esse negócio"
3. Receive: Decision Question + Mermaid + scored GAP list + executive synthesis
```

---

## Main outputs

| Output | Description |
|--------|-------------|
| Decision Question | Single binary question the business must answer |
| Mermaid diagram | System or flow map of the business model |
| GAP/BM/RISCO IDs | Scored items: Impact × Urgency × Confidence × Feasibility |
| Executive synthesis | ≤ 300 words, fact/hypothesis separated |
| Canonical pipeline | Phases S00–S20, Gates G0–G12 (v3) |
