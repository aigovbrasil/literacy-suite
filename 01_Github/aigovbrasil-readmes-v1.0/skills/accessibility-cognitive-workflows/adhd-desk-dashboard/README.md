# adhd-desk-dashboard · ADHD-Friendly Workflow Dashboard

**Category:** Accessibility & Cognitive Workflows · **Version:** current · **Language:** PT-BR
**Owner:** Leonardo Batista

> ⚠️ **Sensitivity flag R-004:** This skill is designed for users with ADHD/TDAH and dyslexia. Output may reference cognitive accessibility needs. Use with appropriate care and professional judgment. This skill does not provide medical advice.

## What it does
Creates low-cognitive-load printable workflow dashboards and Linear backlog plans for solo knowledge workers with ADHD/TDAH and dyslexia constraints. Output: A4 landscape PPTX deck, PDF, markdown checklists, and structured Linear backlog.

## Activate when
```
ADHD-friendly workflows          printable dashboards
desk operating systems           paper-OS setups
cognitive-load reduction         accessibility-oriented layouts
daily execution templates        sprint cards
"mesa de trabalho"               "painel de operação"
```
Also activates for requests involving Linear + calendar + paper stack orchestration, A4 printable templates, or any combination of physical/digital workflow design — even without explicit ADHD mention.

## Output formats
- A4 landscape PPTX (print-ready)
- PDF version
- Markdown checklists
- Linear backlog structure

## Design principles applied
- High contrast · large type · one task per card · no visual clutter · clear priority signals
