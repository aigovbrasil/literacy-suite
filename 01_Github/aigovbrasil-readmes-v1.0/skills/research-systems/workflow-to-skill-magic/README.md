# workflow-to-skill-magic · Workflow → Skill Converter

**Category:** Research Systems · **Version:** current · **Language:** PT-BR
**Owner:** Leonardo Batista

## What it does
Transforms casual Claude workflows into professional, reusable skills with a visual interactive widget. Optimized for non-developers — 3 questions only, instant visual preview, documentation in PT-BR, zero technical jargon.

## Activate when
```
"transforma isso em skill"      "quero automatizar isso"
"cria skill desse workflow"     "skill-ify this"
"vira skill"
```
Also activates when user describes a repetitive workflow (e.g., "every Monday I create the weekly schedule").

## 3-step process
```
1. Describe your workflow (what you do repeatedly)
2. Answer 3 questions: What? Who? When?
3. Receive: visual skill widget preview + SKILL.md ready to upload
```

## Integrates with
`live-prompt-pro-converter` (normalization) · `cmd-02-mirp` (full pipeline) · `skill-publish-and-register` (Notion registry)

## Best for
Non-technical users who have a recurring Claude workflow and want to package it as a reusable skill
