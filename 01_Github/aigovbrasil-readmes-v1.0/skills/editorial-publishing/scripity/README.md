# Scripity · Business Document Corpus Generator

**Category:** Editorial Publishing · **Version:** current · **Language:** PT-BR
**Owner:** Leonardo Batista

## What it does
Generates 17 canonical business artifacts (A01–A17) + analysis of 19 frameworks (F1–F7) in gold-standard format from any product or person metadata. Produces a complete document corpus in one run.

## Activate when
```
"novo corpus"                    "gerar corpus para [name]"
"rodar scripity"                 "criar 17 artefatos"
"gerar corpus de [produto]"      "scripity para [pessoa]"
"corpus completo de [produto]"   "quero gerar meu corpus"
"gerar A01 a A17"               "montar corpus padrão-ouro"
```
Also activates when user pastes a YAML metadata block and requests corpus generation.

## 17 artifacts (A01–A17)
Includes: product brief, MRD, BRD, PRD, positioning statement, ICP definition, messaging matrix, GTM plan, content calendar, social media kit, email sequences, FAQ, objection handling, partner deck, investor narrative, roadmap, and launch checklist.

## Input
YAML metadata block with product/person details. The more complete the input, the higher the output quality.

## Do NOT activate for
Isolated single-artifact requests · generic documentation without product context
