# live-prompt-pro-converter · Prompt Optimizer

**Category:** Editorial Publishing · **Version:** current · **Language:** PT-BR / EN
**Owner:** Leonardo Batista

## What it does
Converts, restructures, and optimizes prompts for maximum effectiveness. Works with prompts in any language. Default target model: Claude (Anthropic). Adapts for GPT/Gemini/others if specified.

## Activate when
```
"converta esse prompt"       "melhore esse prompt"
"otimize esse prompt"        "reestruture"
"prompt engineering"         "transforma isso num prompt bom"
"rewrite this prompt"        "make this prompt better"
"estruture esse pedido"      "formata esse prompt"
```
Also activates when user pastes a prompt and asks for analysis or feedback.

## What it produces
- Restructured prompt with clear role, context, task, format, and constraints
- Before/after comparison
- Explanation of changes made
- Optional: variants for different models

## Best for
Long complex prompts that produce inconsistent outputs · prompts that need persona injection · prompts for repeatable workflows
